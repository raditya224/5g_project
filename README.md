# 5G Core Network Deployment
This project involves automating the installation of Docker and the configuration of Open5GS in a Proxmox virtual environment
## Tech Stack
* **Virtualization :** Proxmox VE (LXC Container)
* **OS :** Ubuntu 22.04.5 LTS (Jammy Jellyfish)
* **Containerization :** Docker & Docker Compose
* **Core Network :** Open5GS
##  Installation steps
First, clone this repository to your server or local computer :
```bash
git clone https://github.com/raditya224/5g_project.git
cd 5g_project
```
If you don't have Docker installed yet, install it automatically using the following command : 
```bash
chmod +x docker.sh
./docker.sh
```
